About Me

I have a deep understanding of modern web technologies like JavaScript (React, Node.js), HTML, CSS, and databases (SQL, NoSQL), , I take pride in writing clean, maintainable code that not only works but works well. My goal? To make the web a better place, one project at a time. From tackling performance optimization to ensuring responsive designs, I’m constantly learning and pushing the boundaries of what’s possible on the web

I can build dynamic web applications by integrating React or Vue on the front end with a Node.js backend. I would connect my app to a database (such as MongoDB or MySQL) and use S3 for handling media storage. I might also implement authentication (using JWT, for example) to manage user sessions.
